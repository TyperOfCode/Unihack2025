# Get backend setup:


## Initialize UV
Download UV: https://docs.astral.sh/uv/getting-started/installation/

```
curl -LsSf https://astral.sh/uv/install.sh | sh
```

## do UV sync

```
uv sync
```

## source the venv

```
source .venv/bin/activate
```

## run the backend

```
uv run fastapi dev
```
